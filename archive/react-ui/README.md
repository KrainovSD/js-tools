# @krainovsd/react-ui

The library of React-UI components based on [Ant Design](https://ant.design/).

## Installing

### Package manager

Using pnpm:
```
pnpm install @krainovsd/react-ui
```

Using yarn:
```
yarn add @krainovsd/react-ui
```

Using npm:
```
npm install @krainovsd/react-ui
```


## Usage

```js
import { Flex } from "@krainovsd/react-ui"

<Flex>text</Flex>
```
